# 施夷光 — QQ 智能机器人

基于 Claude Code + cc-connect + NapCat 搭建的王者荣耀「西施」角色扮演 QQ 机器人。施夷光是她的本名，大家都叫她西施。

## 项目概览

```
QQ 用户 → NapCat QQ 客户端 ──→ Python 代理 (3002) ──→ cc-connect ──→ Claude Code
                                      ↑                      ↑              ↑
                                 过滤杂讯/排队提醒      多项目管理       AI 大脑
                                                       Telegram 编程助手  西施人格
```

## 启动流程

### 1. 启动 VPN（ikuu）

确保代理 `127.0.0.1:7890` 可用。

### 2. 启动 NapCat QQ

双击 `E:\xishi-bot\NapCat\start_qq.bat`，用小号扫码登录。

二维码位置：`E:\xishi-bot\NapCat\NapCat.44498.Shell\versions\9.9.26-44498\resources\app\napcat\cache\qrcode.png`

也可通过 WebUI 查看：`http://127.0.0.1:6099/webui?token=`（token 见控制台输出）

### 3. 启动 Python 代理

```powershell
python E:\xishi-bot\qq-proxy\proxy.py
```

### 4. 启动 cc-connect

```powershell
$env:HTTP_PROXY="http://127.0.0.1:7890"
$env:HTTPS_PROXY="http://127.0.0.1:7890"
E:\xishi-bot\cc-connect\cc-connect.exe --config E:\xishi-bot\cc-connect\config.toml
```

## 修改人设

编辑 `E:\xishi-bot\xishi-bot\CLAUDE.md`，然后在 QQ 给小号发 `/new` 即可生效，无需重启。

## 已知限制

- 连续发两条消息会触发排队提醒（代理已拦截，但偶尔漏网）
- QQ 非官方协议，NapCat 可能随 QQ 更新失效
- Telegram API 需走代理
